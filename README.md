Project Overview:-

This project analyzes a dataset of 103,024 Ola bookings to identify the root causes of a 28.08% ride cancellation rate. 
By leveraging Excel, MySQL, and Power BI, I identified key behavioral and operational bottlenecks and proposed a 
data-driven strategy to reduce cancellations by 10%, potentially recovering thousands of lost bookings.


Tech Stack & Tools::-

Excel: Initial data cleaning, duplicate removal, and data structural adjustments.

MySQL: Large-scale data management, exploratory data analysis (EDA), and creating business-logic views.

Power BI: Interactive dashboarding and geospatial trend visualization.


How to Run this Project::-

Data Cleaning: Open the dataset in Excel to view the duplicate removal and data formatting steps.

Database Setup: Run the scripts in SQL_Scripts/ using MySQL Workbench to load the data and create the necessary views.

Visualization: Open the ola_booking.pbix file in Power BI Desktop to interact with the dashboard.
